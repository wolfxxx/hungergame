# Neon Pac

A Pac-Man style game built with Phaser 3.

## Gameplay

- Use the Arrow keys or WASD to move.
- Eat all the dots to complete a level.
- Power orbs allow you to eat ghosts for a short time.
- Fruits appear for bonus points and a speed boost.
- Go through the tunnels to appear on the other side of the screen with a "woosh" sound and a speed boost.

## Controls

- **Arrow Keys / WASD:** Move
- **M:** Mute/Unmute
- **P:** Pause/Unpause
- **R:** Restart

## Levels

The game features 6 levels with different layouts and increasing difficulty:
1.  Classic
2.  Corridors
3.  Labyrinth
4.  Speedway
5.  Fortress
6.  Chaos
